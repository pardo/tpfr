# Tasker
Tasker, TODO app para la cursada de TTPS 2016
## Requisitos
```
rails 5.x
ruby 2.3.x
bundler
```
## Instalacion
```bash
git clone https://github.com/pardo/tpfr
$ cd tpfr/
$ bundle install
# set intial db setup
$ rails db:migrate
$ rails db:seed
# start server
$ rails server
```
## Test
Para correr los test, desde la carpeta en la que clonamos el proyecto ejecutar
```
$ rails test
```
